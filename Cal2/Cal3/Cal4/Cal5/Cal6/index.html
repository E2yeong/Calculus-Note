Understanding Improper Integrals, Growth Models, and Convergence

Chapter 7.8 – Improper Integrals (pp.560–573)

1️⃣ What Are Improper Integrals?

An improper integral deals with cases where the interval is infinitely long, or where the function becomes infinite (blows up).

Even when something seems “infinite,” the total area might still approach a finite value — we use limits to check if that happens.

Case 1 – Infinite Interval

∫₁^∞ (1/x²) dx = lim(b→∞) [-1/x]₁^b = 1

Although the graph stretches forever, the tail gets smaller so fast that the total area converges to 1.

This is called a convergent improper integral — it has a finite result.

Case 2 – Divergent Example

∫₁^∞ (1/x) dx = lim(b→∞) [ln x]₁^b = ∞

The function 1/x decreases slowly, so the area keeps growing without limit — it diverges.

Case 3 – Function Blows Up Inside the Interval

∫₀¹ (1/√x) dx = lim(a→0⁺) ∫ₐ¹ x^(-1/2) dx = [2√x]ₐ¹ = 2

Even though it becomes infinite at x = 0, the total area is still finite → convergent.

- Infinite limits or singular points don’t always mean “infinite area.” - Convergent → finite area, Divergent → infinite area.

Chapter 8 – Differential Equations (574–606)

2️⃣ What Are Differential Equations?

A differential equation involves derivatives — it describes how a quantity changes rather than what it is. Solving it means finding the original function that fits that rate of change.

Example: dy/dx = 3x² → Integrate both sides → y = ∫3x² dx = x³ + C The result is called the general solution.

Separable Differential Equations

Equations that can be “split” into x terms and y terms:

dy/dx = g(x)·h(y) → dy/h(y) = g(x) dx

Then integrate both sides.

Example: dy/dx = x·y → (1/y) dy = x dx → ln|y| = x²/2 + C → y = Ce^(x²/2)

3️⃣ Exponential Growth and Decay (8.1)

Change is proportional to the current amount:

dy/dt = k·y
  • k > 0 → exponential growth
  • k < 0 → exponential decay

Integrating both sides:

dy/y = k dt → ln|y| = kt + C → y = Ce^(kt)

This describes population growth, radioactive decay, and cooling processes.

4️⃣ Logistic Growth (8.2)

dy/dt = k·y·(1 − y/M)
  • M = maximum capacity (carrying capacity)
  • k = growth rate

Starts exponentially, slows down as it nears the limit M → forming an S-shaped curve (sigmoid).

Chapter 9 – Sequences and Series (608–641)

5️⃣ Sequences

A sequence is an ordered list of numbers following a pattern.

Example: 1, 1/2, 1/3, 1/4, … → aₙ = 1/n

Convergence and Divergence

  • If aₙ → a specific number → the sequence converges.
  • If it doesn’t settle on any number → it diverges.
aₙ = 1/n → 0 (convergent)
aₙ = (−1)ⁿ → oscillates (divergent)

6️⃣ Series

A series is the sum of all terms in a sequence:

S = a₁ + a₂ + a₃ + …

Geometric Series

Form: a + ar + ar² + … Convergent if |r| < 1

S = a / (1 − r)
Example: 1 + 1/2 + 1/4 + 1/8 + … = 2

Divergence Test

If lim (n→∞) aₙ ≠ 0 → the series diverges.

If terms don’t approach 0, the sum keeps growing forever.

Ratio Test

L = lim (n→∞) |aₙ₊₁ / aₙ|
  • L < 1 → convergent
  • L > 1 → divergent
  • L = 1 → inconclusive

7️⃣ Key Takeaways

  • Improper Integrals: some “infinite” areas can still be finite.
  • Differential Equations: find functions from their rate of change.
  • Exponential Models: describe growth, decay, and real-world change.
  • Sequences: study how numbers behave as n → ∞.
  • Series: determine if infinite sums converge or diverge.
