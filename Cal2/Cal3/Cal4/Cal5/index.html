Chapter 6 – Applications of Integration (467 – 491)

6.4 Work

If force is constant:

W = Force × Distance

If force varies:

W = ∫ab F(x) dx
Example (Hooke’s Law):
F(x) = k x → Work = ∫₀ᴸ k x dx = (k L²)/2

The integral represents the total work as the sum of infinitesimal (force × distance) elements.

6.5 Average Value of a Function

favg = (1 / (b − a)) ∫ab f(x) dx

The “average height” of a function equals the area under the curve divided by the interval length.

Example:
f(x) = x² on [0, 2]
∫₀² x² dx = 8/3 → favg = (1/2)(8/3) = 4/3

Average value concepts are often applied in real data analysis and physical measurements.

Chapter 7 – Techniques of Integration (492 – 542)

7.1 Integration by Parts

∫ u dv = u v − ∫ v du

Choose u → differentiates easily, dv → integrates neatly.

Example: ∫ x eˣ dx
u = x, dv = eˣ dx → du = dx, v = eˣ
⇒ x eˣ − ∫ eˣ dx = (x − 1) eˣ + C

7.2 Trigonometric Integrals

Integrals of the form ∫ sinᵐx cosⁿx dx.

  • If one power is odd → separate one term for substitution.
  • If both even → use half-angle identities.
∫ sin³x dx = ∫ (1 − cos²x) sinx dx = −cosx + (1/3) cos³x + C

7.3 Trigonometric Substitution

Use trig substitution when radicals appear:

FormSubstitution
√(a² − x²)x = a sinθ
√(a² + x²)x = a tanθ
√(x² − a²)x = a secθ
Example: ∫ dx / √(1 − x²)
Let x = sinθ → dx = cosθ dθ → ∫ cosθ / cosθ dθ = θ = arcsin x + C

7.4 Partial Fractions

For rational functions, decompose before integrating.

∫ (2x + 3)/(x² + x) dx = ∫ (A/x + B/(x + 1)) dx = 2 ln|x| + ln|x + 1| + C

7.5 Integration Strategy

  1. Try basic formulas.
  2. Use substitution.
  3. Integration by parts.
  4. Trigonometric integrals / substitution.
  5. Partial fractions.
  6. Numerical approximation if needed.
Practice identifying which method fits best — a key exam skill.

7.6 Using Tables & CAS

Complex integrals may rely on integration tables or computer algebra systems (CAS). AP exams distinguish between formula-memorization and CAS-based reasoning.

7.7 Approximate Integration

Numerical Methods:

  • Midpoint Rule
  • Trapezoidal Rule
  • Simpson’s Rule
Example: ∫₀² x² dx, Trapezoidal Rule (n = 2)
(Δx / 2)[f(0) + 2 f(1) + f(2)] = 1 × [0 + 2(1) + 4] = 6
True value = 8/3 ≈ 2.67 → approximation = 3 → check error.
